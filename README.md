# InkBall overview

The object of Microsoft® InkBall is to draw ink strokes with your tablet pen to guide colored balls into holes of the same color.

The amount of points that you earn is based on the color of the ball. The game ends when a ball enters a hole of a different color or the game timer runs out.

In InkBall, gray is a neutral color. Gray balls can enter a hole of any color, and a gray hole can accept a ball of any color. However, no points are awarded.

By choosing from a variety of game boards and five levels of difficulty, you can make InkBall as challenging as you want.
